# 하랑 한국사 개념별 이미지 프롬프트

개념 이미지는 총 33장이다. 생성할 때 현재 완성된 메인 캐릭터 이미지를 매번 참고 이미지로 첨부한다.

## 공통 프롬프트

    첨부한 픽셀 아트 강아지 캐릭터를 기준 캐릭터로 사용한다.

    흰색 강아지, 길게 늘어진 귀, 보라색 점눈과 둥근 코, 빨간 전통 모자, 청록색 옷, 짧고 통통한 2등신 체형을 정확히 유지한다. 얼굴 형태, 귀 길이, 색상 팔레트, 외곽선 굵기와 픽셀 크기도 참고 이미지와 동일하게 유지한다.

    귀여운 16비트 RPG 게임 스프라이트 스타일. 64×64 스프라이트를 확대한 듯한 명확한 픽셀 격자, 안티앨리어싱 없는 단단한 가장자리, 제한된 색상 팔레트, 2단계 셀 셰이딩. 벡터 선, 부드러운 그라데이션, 3D 표현을 사용하지 않는다.

    개념 카드에 사용할 정사각형 1:1 구도. 캐릭터가 화면의 약 65%를 차지하도록 중앙 배치한다. 역사 소품은 핵심적인 것 1~2개만 사용하고 캐릭터보다 크게 만들지 않는다. 캐릭터와 소품이 잘리지 않도록 여백을 둔다.

    실제 알파 채널이 적용된 완전한 투명 배경 PNG. 흰색이나 회색 체커보드 무늬를 이미지에 그리지 않는다. 배경 풍경, 글자, 숫자, 로고, 워터마크 없음.

공통 제외 프롬프트:

    tall body, long legs, vector illustration, smooth outline, gradient, glossy shading, realistic anatomy, 3D render, detailed background, checkerboard background, text, watermark, oversized props, complex costume

## 생성 순서

한 번에 모두 만들지 말고 아래 순서로 생성한다.

1. 선사·고조선 5장
2. 여러 나라 4장
3. 삼국·가야 5장
4. 남북국 5장
5. 후삼국 4장
6. 고려 10장

각 묶음의 첫 결과에서 캐릭터 비율, 외곽선, 팔레트가 기준 이미지와 같은지 확인한 뒤 나머지를 생성한다.
