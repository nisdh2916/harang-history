# 하랑의 한국사 탐험대

중학교 3학년 시험 범위인 선사 시대부터 고려 시대까지 학습하는 개인용 한국사 사이트입니다.

## 바로가기

- 학습 사이트: https://nisdh2916.github.io/harang-history/
- 시대별 개념 노트 33개
- 오늘의 10문제, 전체 문제, 시대별 문제, 오답 복습

진도와 오답은 사용하는 브라우저에 저장됩니다.

## 구성

- index.html: 시대별 메인 화면
- era.html: 시대별 개념 목록
- note.html: 개념 암기 노트
- quiz.html: 한국사 퀴즈

이미지는 assets/characters, 학습 내용은 assets/data.js와 assets/scope-goryeo.js에서 관리합니다.
